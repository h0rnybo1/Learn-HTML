HTML Forms Tutorial

Overview

Forms are a crucial part of web development, enabling user interaction and data collection. This guide will walk you through the basic structure of an HTML form.

Form Tags Explanation

  • <form>: Used to create an HTML form for user input. Contains all the form elements.
  • <input>: Used to create various types of input fields, such as text, radio buttons, checkboxes, etc.
  • <label>: Defines a label for an <input> element, improving accessibility and usability.
  • <textarea>: Creates a multi-line text input field.
  • <button>: Defines a clickable button.
  • <select>: Creates a drop-down list.
  • <option>: Defines an option in a drop-down list.

Simple HTML Form Code


    <!DOCTYPE html>
    <html>
    <head>
        <title>Basic Form</title>
    </head>
    <body>
    
        <form method="GET">
            <label for="username">Username:</label><br>
            <input type="text" id="username" name="username" required><br><br>
    
            <label for="email">Email:</label><br>
            <input type="email" id="email" name="email" required><br><br>
    
            <label for="comments">Comments:</label><br>
            <textarea id="comments" name="comments" rows="4" cols="50"></textarea><br><br>
    
            <input type="submit" value="Submit">
        </form>
    
    </body>
    </html>
