Formatting in HTML

HTML provides different way to format tags

HTML formatting elements

Formatting tags in HTML are:

  • <b>- Bold text
  • <i> - italic text
  • <u> - underlined text
  • <del> - striked through text
  • <mark> - highlighted text
  • <small> - smaller text
  • <sup> - smaller text
  • <sub> - smaller text

HTML <b> Element

The <b> tag is used in HTML to make text bold. It is often used to emphasize a particular word or phrase.

Example:

Here is an example of how to use the <b> tag:

This is a bold word in a sentence.

HTML <i> Element

The <i> tag is used in HTML to make text italic. It is often used to indicate technical term, a phrase from another language, a thought, a ship name, etc.

Example:

Here is an example of how to use the <i> tag:

This is a italic word in a sentence.

HTML <u> Element

The <u> tag is used in HTML to make text underlined. It is often used to focus on a particular word or phrase.

Example:

Here is an example of how to use the <u> tag:

This is a inderlined word in a sentence.

HTML <del> Element

The <del> tag is used in HTML to make text striked out. It is often used to correct or remove certain word or phrase.

Example:

Here is an example of how to use the <del> tag:

Go buy books from the stationary stationery shop

HTML <mark> Element

The <mark> tag is used in HTML to make text highlighted. It is often used to emphasize a particular word or phrase.

Example:

Here is an example of how to use the <mark> tag:

Alas! she was dead!

HTML <small> Element

The <small> tag is used in HTML to make text smaller. It is often used to make certain words smaller.

Example:

Here is an example of how to use the <small> tag:

It's is a small world after all.

HTML <sup> Element

The <sup> tag is used in HTML to make text superscripted. It is often used to denote mathematical exponents.

Example:

Here is an example of how to use the <sup> tag:

p2 + b2 = h2

HTML <sub> Element

The <sub> tag is used in HTML to make text subscripted. It is often used to denote chemical molecules.

Example:

Here is an example of how to use the <sub> tag:

H2+ 2O2 = H2O
