Basics of HTML

HTML Elements

HTML documents are made up of elements, which consist of a start tag, content, and an end tag. Elements can also have attributes that provide additional information.

Common HTML Tags

Here are some of the most commonly used HTML tags:

  • <h1> to <h6> - Heading tags, with <h1> being the highest level and <h6> the lowest.
  • <p> - Paragraph tag used to define blocks of text.
  • <a> - Anchor tag for creating hyperlinks.
  • <img> - Tag used to embed images.
  • <ul> - Unordered list, typically rendered as bullet points.
  • <ol> - Ordered list, typically rendered as numbered items.
  • <li> - List item used within <ul> or <ol>.
  • <div> - Division tag used as a container for other elements to create sections.
  • <span> - Inline container used to wrap text or other inline elements.
  • <strong> and <em> - Tags used to emphasize text (bold and italicize).

Attributes

Attributes provide additional information about HTML elements. They are always included in the opening tag and typically come in name/value pairs like name="value".

  • href - Specifies the URL of the page the link goes to in an <a> tag.
  • src - Specifies the path to the image in an <img> tag.
  • alt - Provides alternative text for an image in case it cannot be displayed.
  • width and height - Define the width and height of an image.
  • target - Specifies where to open the linked document.

Document Object Model (DOM)

The DOM is a representation of the HTML document as a tree structure. It allows programming languages like JavaScript to manipulate the content, structure, and style of a website dynamically.

HTML5

HTML5 is the latest version of HTML and includes new elements and attributes to support modern web development. Some of the new features include:

  • <header>, <footer>, <section>, <article>, <aside> - Semantic elements that define different parts of a webpage.
  • <audio> and <video> - Tags for embedding multimedia content.
  • <input> with types like date, email, range, etc. - New form controls for better user input experiences.
