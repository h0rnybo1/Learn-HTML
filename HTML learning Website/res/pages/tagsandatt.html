Introduction to HTML Tags and Attributes

What are HTML Tags?

HTML tags are the building blocks of an HTML document. They create the structure and content of a webpage.

Common HTML Tags

  • <html>: The root element of an HTML page.
  • <head>: Contains meta-information about the document.
  • <title>: Sets the title of the document, shown in the browser title bar.
  • <body>: Contains the content of the document, such as text, images, links, etc.
  • <h1> to <h6>: Header tags, with <h1> as the largest and <h6> as the smallest.
  • <p>: Defines a paragraph.
  • <a>: Creates a hyperlink.
  • <img>: Embeds an image.
  • <ul> and <li>: Create an unordered list and list items.
  • <div>: A container for other elements.

What are HTML Attributes?

Attributes provide additional information about elements and are always included in the opening tag.

Common HTML Attributes

  • href: Specifies the URL for a link. full form: Hyperlink Refernce
  • src: Specifies the path to an image. full form: source
  • alt: Provides alternative text for an image.
  • title: Adds extra information displayed as a tooltip when you hover over the element.
