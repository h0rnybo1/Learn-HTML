Introduction to HTML

What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages and web applications. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

Basic Structure of an HTML Document

An HTML document consists of several essential elements:

  • <!DOCTYPE html> - Declares the document type and version of HTML.
  • <html> - The root element that contains all other HTML elements.
  • <head> - Contains meta-information about the document, such as its title and links to stylesheets.
  • <title> - Sets the title of the document, which appears in the browser's title bar or tab.
  • <body> - Contains the visible content of the document, such as text, images, and links.

History of HTML

HTML was created by Tim Berners-Lee in 1991, but it wasn't widely adopted until the release of HTML 2.0 in 1995. Since then, HTML has evolved significantly, with HTML5 being the latest standard as of this writing. HTML5 introduces new elements, attributes, and behaviors, providing more powerful and flexible ways to build web applications.

Why Learn HTML?

Learning HTML is essential for anyone who wants to create web pages. Understanding HTML allows you to:

  • Create structured documents with headings, paragraphs, lists, and more.
  • Embed images, videos, and other multimedia elements.
  • Link to other web pages and resources.
  • Form the foundation for learning other web technologies like CSS and JavaScript.

Common HTML Tags

Here are some of the most commonly used HTML tags:

  • <h1> to <h6> - Heading tags, with <h1> being the highest level and <h6> the lowest.
  • <p> - Paragraph tag.
  • <a> - Anchor tag for creating hyperlinks.
  • <img> - Tag for embedding images.
  • <ul> - Unordered list.
  • <ol> - Ordered list.
  • <li> - List item.
  • <div> - Division tag used as a container for other elements.
  • <span> - Inline container used to wrap text or other inline elements.
