Headings and Paragraphs

HTML provides six levels of headings, <h1> to <h6> , with <h1> being the highest (or most important) level and <h6> the least.

The <p> tag defines a paragraph of text. Each paragraph will have some space before and after it to separate it from other content.

This is a paragraph.

The <br> tag is used for line breaks. For example:

This is line one.
This is line two.

Text Formatting

HTML provides various tags to format text. Here are a few common ones:

  • <b> : Makes text bold. This is bold text.
  • <strong> : Also makes text bold, but with added importance. This is strong text.
  • <i> : Italicizes text. This is italic text.
  • <em> : Italicizes text with emphasis. This is emphasized text.
  • <mark> : Highlights text. This text is highlighted.
  • <del> : Shows deleted text. This text is deleted.
  • <ins> : Shows inserted text. This text is inserted.
