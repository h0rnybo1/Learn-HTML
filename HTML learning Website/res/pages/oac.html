Opening and Closing Tags in HTML

What are HTML Tags?

HTML tags are the building blocks of HTML. They are used to create elements in a webpage, and most tags come in pairs: an opening tag and a closing tag.

Opening Tags

An opening tag is the starting point of an HTML element. It is enclosed within angle brackets, and it can include attributes that provide additional information about the element. For example:

  • <p> - This is the opening tag for a paragraph element.
  • <a href="https://www.example.com"> - This is the opening tag for a hyperlink with an attribute specifying the URL.
  • <img src="image.jpg" alt="A descriptive text"> - This is the opening tag for an image element with attributes for the source file and alternative text.

Closing Tags

A closing tag is the end point of an HTML element. It is similar to an opening tag but includes a forward slash before the tag name. Not all HTML tags require a closing tag (e.g., <img> and <br> are self-closing). Examples of closing tags:

  • </p> - This is the closing tag for a paragraph element.
  • </a> - This is the closing tag for a hyperlink element.
  • </div> - This is the closing tag for a division element.

Examples and Uses

Let's look at some examples of how opening and closing tags are used to create HTML elements:


<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Document</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph of text.</p>
    <a href="https://www.example.com">Visit Example.com</a>
    <img src="image.jpg" alt="Description of the image">
</body>
</html>
